## One Time Message 
This is a simple one time message app. It is a simple app that allows you to send a message to a user and they can only view it once. Once the message is viewed, it is deleted from the database.

### Project Stack
- Frontend: 
  - react
  - shadcn
  - superagent
  - react-router
- Backend:
  - bun
  - elysia
  - sqlite
